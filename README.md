# FinDash - Personal Finance Dashboard

FinDash is a modern, premium-quality Personal Finance Dashboard designed to help users manage their financial activities with elegance and ease. Built with a focus on UI/UX, it features a glassmorphism design aesthetic, responsive layout, and interactive data visualizations.

## ğŸ“¸ Views

### 1. Dashboard (Overview)

The central hub displaying:

- **My Cards**: specific credit/debit cards with details.
